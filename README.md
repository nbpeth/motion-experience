## Setup
### Automated Setup
* Install Python > 3.14.6
* Run `$ provision.sh`:

### Manual Setup
#### Setup python virtual environment
    * `$ python3 venv -m ./venv`
#### Active virtual environment
    * `$ source ./venv/bin/activate`
#### Install Dependencies
    * `$ pip3 install -r requirements.txt`


## Run
#### Start web app
* `python3 -m http.server 8000`
